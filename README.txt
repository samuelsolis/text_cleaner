This is a script that should be executed with php-cli in Terminal.

Functions: 
  csvWorker class. You can pass input and output folders by param.
  
  function test(). Test if the class is loaded correctly and the input and output folder can be read.

  function init(). Scan input directoy and read the name of the files inside.

  function work(). Process the elements inside input folder.

  function set_debug(true). By param enable or disable debug mode. Thsi mode show some messages in the Console.
